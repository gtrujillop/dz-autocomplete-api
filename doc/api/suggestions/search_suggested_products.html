Suggestions API

Search suggested products

GET /suggestions

This method returns a JSON with products based on query params.

Parameters

Name Description
q Querystring param for search terms by product name
max_price Max product price limit
min_price Min product price limit
cat Category for product

Request

Headers

Host: example.org
Cookie:

Route

GET /suggestions?q=a&min_price=1&max_price=500&cat=129

Query Parameters

q: a
min_price: 1
max_price: 500
cat: 129
